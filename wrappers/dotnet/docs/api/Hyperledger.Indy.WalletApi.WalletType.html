    Parameters
    Type Name Description
    String name

    The name of the wallet to open.
    String config

    The configuration for the wallet.
    String runtimeConfig

    The runtime configuration for the wallet.
    String credentials

    The credentials of the wallet.
    Int32 walletHandle

    The handle assigned to the wallet.
    Returns
    Type Description
    ErrorCode

    An ErrorCode value indicating the outcome of the operation.
    Remarks

    When implementing a custom wallet this method is responsible for opening the wallet and returning a handle for the opened wallet. The value of the runtimeConfig parameter should override any corresponding values provided in the config parameter and value of the credentials parameter should be used to control access to the wallet.
